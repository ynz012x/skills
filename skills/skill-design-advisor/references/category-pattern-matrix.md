## 模式 → 实践的天然关联

某些实践与特定模式有天然亲和，无论类别如何：

| 模式 | 天然关联的实践 | 原因 |
|------|-------------|------|
| Tool Wrapper | #3 渐进披露 | 渐进披露就是该模式的核心理念 |
| Generator | #4 不过度约束 | 模板必须留出填充空间 |
| Reviewer | #2 Gotchas 清单 | 检查清单从 gotchas 演化而来 |
| Inversion | #5 初始化需求 | Inversion 本质就是关于初始化 |
| Pipeline | #7 数据持久化 | 长流程需要状态持久化 |

**选择逻辑**：取类别推荐（3 条）和模式关联（1 条）的并集，去重后保留 3-5 条。
